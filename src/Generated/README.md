# Generated

This directory is auto-generated by `npm run generate`. Do not hand edit.

The generator reads `https://roxyapi.com/api/v2/openapi.json` and emits:

- `Client.php`: typed PHP method per endpoint
- `Endpoints.php`: endpoint registry with TTLs and per endpoint attribute maps
- `Domains.php`: brand-domain registry derived from `bin/domains.json`
- `Shortcodes/*.php`: one shortcode class per non hero endpoint
- `Heroes/*.php` and `Heroes/Manifest.php`: hero shortcode classes built from `bin/hero-config.json`
- `Forms/*.php`: form-mode specs for endpoints whose request body takes a required nested object

CI fails if the contents of this directory drift from the live OpenAPI spec. To resolve drift:

```bash
npm run generate
git add src/Generated blocks/generated
git commit -m "regenerate from openapi"
```

To change a hero, edit `bin/hero-config.json`. To patch a stale upstream example value, edit `bin/example-overrides.json`. For anything else, edit `bin/generate.mjs` and regenerate.
